Info table


Since an Info Graph consists of Info Triples and Info Triples consists of three vertices, an Info Graph can be written in a simple table format with three columns which we call an Info Table. The first column in the Info Table is for the primary key which is the identifier of the Edge Vertex. The other columns holds the identifiers of the adjacent vertices of the Info Triple.
